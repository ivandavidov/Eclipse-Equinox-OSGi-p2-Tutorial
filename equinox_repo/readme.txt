1) Download Equinox OSGi p2 repository:

   http://download.eclipse.org/equinox/drops/R-Mars.1-201509040015/index.php   

2) Extract the Equinox OSGi p2 repository in the current folder.
   You should have the following directory structure:

      equinox_osgi_p2_tutorial  (root project folder)
      |
      +-- equinox_repo          (current folder)
          |
          +-- binary
          |
          +-- features
          |
          +-- plugins
          |
          +-- artifacts.xml
          |
          +-- content.xml
          |
          +-- readme.txt        (current file)
